🏗️ Architecture Evolution
This portfolio project demonstrates a complete technical evolution across 4 weeks:

Phase 1 (Local CLI): A Python-based terminal application using local JSON data.

Phase 2 (Cloud CLI): Integrated cloud vector synchronization for remote data handling.

Phase 3 (Web App): A professional Next.js 15 application with a modern chat interface, deployed via Vercel.

🚀 Performance & UX Analysis
Speed: Transitioning to Groq Cloud and Upstash Vector provides sub-second response times for complex RAG queries.

UX: The interface includes a model toggle (Fast vs. Detailed) and a mobile-responsive sidebar for seamless use across devices.
